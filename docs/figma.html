Figma: Interactive components & auto layout

Today is about a month since interactive components launched in Figma 👏. In my masters' thesis, I'm creating a concept for one of Hyperlab's customers, Safira. To allow the solution to fit in its natural habitat, I replicated some of the flows of Safira's e-com. While doing so, I took the opportunity to become more efficient with interactive components and auto layout.

Interactive components

Mmmm, the possibility to allow users to actually browse content without creating a huge Figma prototype is so neat. 5/5 toasters 🍞

Auto layout

In the beginning, the auto-layout requires a bit more thinking of how to structure your frames. However, it's almost always worth it in the long run? I like to use it when creating a long page with different sections of content. Auto-layout makes it easy to flip the order around while still being evenly distributed and aligned.
